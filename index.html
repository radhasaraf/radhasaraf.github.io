Standardization of the Shape of Ground Control Point (GCP) and the Methodology for Its Detection in Images for UAV-Based Mapping Applications
Aman Jain, Milind Mahajan, Radha Saraf
CVC(SAI), 2019
Springer

To establish accurate georeferencing techniques for aerial images from UAVs for an accurate earth to orthomap correspondence, we propose the design of a simple marker along with the pipeline for its detection
Panorama stitching for two or more images
project page

Estimated homography between image pairs using feature correspondences. Used Adaptive non-maximal suppression for uniform features and RANSAC for for removing the outliers among the feature matches. Also, trained HomographyNet, a CNN based supervised learning architecture for homography estimation using synthetically generated data.

Collaborative work with Sai Ramana Kiran Pinnamaraju
Probability based edge detection project page

Developed a simplified version of pb, which finds boundaries by examining texture and color discontinuities in addition to intensity discontinuities across multiple scales.

Outperforms Canny and Sobel by suppressing the false positives produced by classical methods in textured regions.
Pose estimation of a mobile robot

Estimated pose of an autonomous mobile robot using differential-RGB color-space, image processing, and the 'SolvePnP' (Perspective-n-point) algorithm from the OpenCV library with an accuracy of +/-5 cms
Kinematic planning under non-holonomic constraints
project page

Developed a kinematic path planner with nonholonomic constraints to efficiently park vehicles ranging from a differential drive bot to a car to a truck with a trailer.
